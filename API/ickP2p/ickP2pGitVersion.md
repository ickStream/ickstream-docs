Get full git commit version for the ickstream p2p library

**Signature:**

``` c
#include <ickP2p.h>

const char *ickP2pGitVersion( void );
```

**Arguments:**

  - none

**Return Value:**

  - String with full git commit number.
  - Example output: "7cd5bf3453623fac497c540275d9c573f1c448c1"
